# Notes captured from shared ChatGPT conversation

## Key concepts extracted
- Daily micro-learning about plant care
- Learning should directly affect companion growth
- Plant companion should talk/react to user actions
- Growth should be visual and emotionally rewarding
- Collection systems and a PlantDex help long-term retention
- Best concept direction was not pure tamagotchi or pure monster game, but a hybrid companion model
- Strongest recommended architecture: reality-linked companion system

## Candidate starter plants
- Monstera
- Basil
- Tomato
- Lavender
- Rosemary

## MVP recommendation from idea thread
- Start small
- Keep sessions short
- Focus on one rewarding loop
- Use 3 plants initially
- Do not build every plant or every system upfront

## UX principles from the idea thread
- Home should be the main hub
- Lesson flow should be very short
- Reward screen matters
- Plant emotion and growth animation matter a lot
